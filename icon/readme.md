<h1 align="center">Alist</h1>

## 这是什么？

一款支持多种存储的目录文件列表程序，后端基于`go-fiber`，前端使用`react`。

## 前端项目地址

- https://github.com/Xhofe/alist-web

## 演示地址

- https://alist.nn.ci


## 支持的存储

- 本地存储
- 阿里云盘
- Onedrive
- ...

## 如何使用

- https://www.nn.ci/archives/alist.html
